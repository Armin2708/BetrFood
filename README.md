# BetrFood - Food Social Media Platform

A clean, TikTok/Instagram-style social media interface designed specifically for food enthusiasts. Built with pure HTML and CSS for maximum simplicity and performance.

## Features

- **Community Feed** - Browse food posts from the community
- **Following Feed** - View posts from people you follow with detailed recipe information
- **Search** - Find recipes, users, and food content with virtual keyboard support
- **Comments** - Engage with posts through comments
- **Share** - Share recipes across multiple platforms
- **Likes** - See who liked your favorite food posts
- **User Profiles** - View user profiles with post grids

## Pages

1. **index.html** - Community feed homepage
2. **following.html** - Following feed with post details
3. **search.html** - Search page with virtual keyboard
4. **comments.html** - Comment section for posts
5. **share.html** - Share modal with multiple platform options
6. **liked-by.html** - List of users who liked a post
7. **profile.html** - User profile with stats and post grid

## Design Features

- Mobile-first responsive design
- Clean, modern iOS-style interface
- Smooth transitions and interactions
- Optimized for 430px mobile viewport
- Pure CSS styling with no frameworks
- Accessible color scheme

## How to Use

1. Open `index.html` in your web browser
2. Navigate between pages using the bottom navigation bar
3. Click on elements to explore different features
4. All pages are interconnected for seamless navigation

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (minimal, only for keyboard interaction)

## Browser Support

Works best in modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)

## Customization

All styling is centralized in `styles.css` with CSS custom properties for easy theming:

```css
:root {
    --primary-color: #007AFF;
    --secondary-color: #5AC8FA;
    --background: #FFFFFF;
    --surface: #F2F2F7;
    --text-primary: #000000;
    --text-secondary: #8E8E93;
}
```
